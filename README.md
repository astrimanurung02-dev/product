# Glow Pekanbaru Landing Page

Ini adalah landing page untuk produk kosmetik di Pekanbaru.

## Cara Menggunakan

1.  Cukup buka file `index.html` di browser Anda (Chrome, Edge, Firefox, dll).
2.  Halaman ini menggunakan **Tailwind CSS** via CDN, jadi pastikan Anda terhubung ke internet agar styling dapat dimuat dengan benar.

## Fitur
-   Desain Premium & Elegan
-   Responsive (Mobile Friendly)
-   Animasi Halus
-   Bagian Produk, Testimoni, dan Kontak
